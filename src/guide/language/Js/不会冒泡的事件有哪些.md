# 不会冒泡的事件有哪些？

## 📋 概述

在 DOM 事件流中，大多数事件都会经历**捕获阶段 → 目标阶段 → 冒泡阶段**。
但有些特殊事件**不会冒泡**，了解这些事件对于事件处理至关重要。

## 🚫 常见的不会冒泡的事件

### 1. UI 事件类
| 事件类型 | 描述 | 示例 |
|---------|------|------|
| **`load`** | 资源加载完成 | `img.onload`, `window.onload` |
| **`unload`** | 文档或资源卸载 | `window.onunload` |
| **`abort`** | 资源加载中止 | `img.onabort` |
| **`error`** | 资源加载失败 | `img.onerror`, `window.onerror` |

### 2. 焦点事件
| 事件类型 | 描述 | 示例 |
|---------|------|------|
| **`focus`** | 元素获得焦点 | `input.onfocus` |
| **`blur`** | 元素失去焦点 | `input.onblur` |

### 3. 鼠标事件（部分）
| 事件类型 | 描述 | 示例 |
|---------|------|------|
| **`mouseenter`** | 鼠标进入元素 | `div.onmouseenter` |
| **`mouseleave`** | 鼠标离开元素 | `div.onmouseleave` |

### 4. 资源事件
| 事件类型 | 描述 | 示例 |
|---------|------|------|
| **`loadstart`** | 资源开始加载 | |
| **`progress`** | 资源加载进度 | |
| **`timeout`** | 资源加载超时 | |

## 🔍 代码示例对比

### 冒泡 vs 不冒泡对比
```html
<div class="parent">
  <input type="text" class="child" placeholder="点击测试焦点事件">
</div>

<script>
const parent = document.querySelector('.parent');
const child = document.querySelector('.child');

// 测试 focus 事件（不冒泡）
child.addEventListener('focus', (e) => {
  console.log('child focus - 不会触发父元素');
});

parent.addEventListener('focus', (e) => {
  console.log('parent focus - 这个不会执行');
});

// 对比：click 事件（会冒泡）
child.addEventListener('click', (e) => {
  console.log('child click');
});

parent.addEventListener('click', (e) => {
  console.log('parent click - 这个会执行');
});
</script>
```

### mouseenter vs mouseover 对比
```html
<div class="outer">
  外部区域
  <div class="inner">内部区域</div>
</div>

<script>
const outer = document.querySelector('.outer');
const inner = document.querySelector('.inner');

// mouseenter/mouseleave（不冒泡）
outer.addEventListener('mouseenter', () => {
  console.log('outer mouseenter');
});

outer.addEventListener('mouseleave', () => {
  console.log('outer mouseleave');
});

// 对比：mouseover/mouseout（会冒泡）
outer.addEventListener('mouseover', () => {
  console.log('outer mouseover - 可能多次触发');
});

outer.addEventListener('mouseout', () => {
  console.log('outer mouseout - 可能多次触发');
});
</script>
```

## ⚠️ 常见问题与解决方案

### 问题1：为什么父元素监听不到 focus 事件？
```javascript
// ❌ 错误做法 - 父元素监听不到
formElement.addEventListener('focus', (e) => {
  // 子 input 获得焦点时不会触发
});

// ✅ 解决方案1：直接在目标元素监听
inputElement.addEventListener('focus', (e) => {
  console.log('input 获得焦点');
});

// ✅ 解决方案2：使用捕获阶段
formElement.addEventListener('focus', (e) => {
  console.log('捕获阶段监听到 focus');
}, true); // 注意第三个参数 true

// ✅ 解决方案3：使用 focusin 事件（会冒泡）
formElement.addEventListener('focusin', (e) => {
  console.log('focusin 会冒泡');
});
```

### 问题2：图片加载错误处理
```javascript
// ❌ 错误做法 - 父元素监听不到 error
container.addEventListener('error', (e) => {
  // 不会触发，因为 error 不冒泡
});

// ✅ 正确做法 - 直接绑定到图片
const img = new Image();
img.addEventListener('error', (e) => {
  console.log('图片加载失败');
  img.src = 'fallback.jpg'; // 备用图片
});
img.src = 'image.jpg';

// ✅ 或者使用捕获阶段
container.addEventListener('error', (e) => {
  if (e.target.tagName === 'IMG') {
    console.log('捕获阶段监听到图片错误');
    e.target.src = 'fallback.jpg';
  }
}, true);
```

## 🔄 替代方案

### 使用会冒泡的对应事件
| 不冒泡事件 | 会冒泡的替代事件 |
|-----------|----------------|
| `focus` | `focusin` |
| `blur` | `focusout` |
| `mouseenter` | `mouseover` |
| `mouseleave` | `mouseout` |

### 事件委托的变通方案
```javascript
// 对于不冒泡的事件，使用捕获阶段实现"反向委托"
document.addEventListener('focus', (e) => {
  if (e.target.matches('.form-input')) {
    console.log('表单输入框获得焦点');
    e.target.classList.add('focused');
  }
}, true); // 使用捕获阶段

document.addEventListener('blur', (e) => {
  if (e.target.matches('.form-input')) {
    console.log('表单输入框失去焦点');
    e.target.classList.remove('focused');
  }
}, true);
```

## 🎯 检测事件是否冒泡

```javascript
function doesEventBubble(eventType) {
  const element = document.createElement('div');
  let bubbles = false;
  
  element.addEventListener(eventType, (e) => {
    bubbles = e.bubbles;
  });
  
  // 触发事件测试
  const event = new Event(eventType);
  element.dispatchEvent(event);
  
  return bubbles;
}

// 测试
console.log('click 冒泡:', doesEventBubble('click')); // true
console.log('focus 冒泡:', doesEventBubble('focus')); // false
console.log('load 冒泡:', doesEventBubble('load')); // false
```

## 💎 总结

### 不会冒泡的事件特点：
- **UI 状态事件**：load、unload、abort、error
- **焦点相关事件**：focus、blur  
- **特定鼠标事件**：mouseenter、mouseleave
- **资源加载事件**：loadstart、progress 等

### 处理策略：
1. **直接绑定**到目标元素
2. **使用捕获阶段**（addEventListener 第三个参数传 true）
3. **寻找替代事件**（如用 focusin 代替 focus）
4. **了解事件特性**，避免错误的事件委托
