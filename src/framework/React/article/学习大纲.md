# 学习大纲

以下大纲基于 React 18/19 与 Next.js 15 最新生态（2025 年），把“从 0 到能独立架构并优化大型项目”拆成 6 个阶段、42 个模块，每个模块给出“核心概念→关键 API→常见坑→实战任务→推荐资源”五步学习法。

---
阶段 1  筑基：JS 现代语法 + React 最小可用知识
1. ES6+ 速通  
   解构、箭头函数、块级作用域、Promise、async/await、模块化、Iterator、Map/Set  
   任务：手写一个「Promise 并发限流器」

2. JSX 与渲染管线
   > [JSX语法与react项目渲染过程 ](https://www.cnblogs.com/FatTiger4399/p/18664207) 
   > [react【框架原理详解】JSX 的本质、SyntheticEvent 合成事件机制、组件渲染过程、组件更新过程](https://developer.aliyun.com/article/1561290)
   编译结果、createElement 签名、jsx-runtime、Babel 配置  
   任务：用 JSX 手写一个 `<Table>` 组件，支持动态列

3. 组件与 Props
   > [组件与Props](/src/framework/React/learn/step1/组件与Props.md)
   函数组件签名、Props 不可变原则、children 切片、prop-types 快速校验  
   任务：把「Antd Button」源码抄一遍，理解二次封装思路

4. State 与事件

   useState 批处理、函数式更新、事件池（React 17 已废弃）与合成事件  
   任务：[实现「双击编辑文本」组件，要求 0 闪烁](/src/framework/React/learn/step1/task/双击编辑文本.md)

5. 列表与 Key  

   diff 策略、key 唯一性、索引坑、React 19 的 BigInt key 支持  
   任务：[实现一个「拖拽排序」列表，性能 < 16 ms](/src/framework/React/learn/step1/task/拖拽排序.md)

6. 条件 & 动态样式  
   三目/&&/|| 陷阱、clsx/classnames、CSS-in-JS 性能对比  
   任务：用 Tailwind 做一个「主题即时切换」按钮

---
阶段 2  核心 Hooks 深度
7. useEffect 生命周期图谱  
   挂载、更新、卸载、依赖数组比较算法、useLayoutEffect 差异  
   任务：实现「窗口尺寸实时获取」自定义 hook，支持 SSR 安全

8. useRef 与“逃生舱口”  
   闭包陷阱、ref 回调、forwardRef、useImperativeHandle  
   任务：[封装「聚焦并全选文字」的 Input 组件](/src/framework/React/learn/step1/task/聚焦并全选文字.md)

9. useMemo/useCallback 精准缓存  
   比较函数、记忆化失效场景、React 编译器自动 memo（React 19）  
   任务：把「千行表格」渲染性能提升 5 倍

10. 自定义 Hook 设计模式  
    单一职责、参数约定、返回值稳定、eslint-plugin-react-hooks  
    任务：写 useDebounce、useInterval、usePrevious 三件套并单测

11. Hooks 规则与源码级理解  
    链表结构、Hook 索引、Dispatcher 切换、Hot-Reload 保持状态  
    任务：用 30 行代码实现迷你 useState

---
阶段 3  单页应用工程化
12. Vite + React 19 脚手架  
    冷启动原理、HMR 边界、环境变量、多入口配置  
    任务：把 CRA 项目迁移到 Vite，构建速度 < 1 s

13. React Router v7（数据路由）  
    createBrowserRouter、loader、action、errorElement、懒加载  
    任务：做一个「带骨架屏」的 Dashboard 路由体系

14. 全局状态管理选型  
    Context 性能陷阱、useContextSelector、Redux Toolkit 切片、Zustand 浅比较、Jotai 原子模型  
    任务：三种方案分别实现「深色模式 + 多语言」开关，对比 FPS

15. 表单双雄：React Hook Form + Zod  
    受控/非受控混合、resolver 集成、实时校验、异步默认选项  
    任务：实现「分步表单」+「动态字段数组」，首屏渲染 < 100 KB

16. 错误边界与兜底  
    componentDidCatch、static getDerivedStateFromError、重置状态、上报 Sentry  
    任务：在路由级别挂 ErrorBoundary，错误恢复不白屏

17. 单元测试策略  
    Jest + React Testing Library、msw 拦截、user-event 行为测试、覆盖率陷阱  
    任务：给自定义 Hook 写 100% 覆盖的单测

---
阶段 4  全栈 & 渲染模式（4 周）
18. Next.js 15 App Router 基础  
    目录即路由、layout、loading、error、parallel routes、intercepting routes  
    任务：把「React Router」项目迁移到 App Router，保持 URL 一致

19. 三种渲染模式对比  
    CSR、SSR、SSG、ISR、PR、React Server Components（RSC）  
    任务：同一博客分别用 SSG 与 ISR 发布，测 Lighthouse 分数差异

20. 数据获取新范式  
    async Server Component、fetch 自动去重、缓存策略（no-store、revalidate、segment）  
    任务：用 RSC + async 组件实现「商品详情」页面，0 客户端请求

21. 中间件与边缘函数  
    NextRequest/NextResponse、A/B 分流、地域重写、Cookie 签名  
    任务：基于地域自动切换货币符号，TTFB < 50 ms

22. 身份验证最佳实践  
    NextAuth v5、Auth0、JWT + HttpOnly、回调路由、服务端鉴权  
    任务：实现「GitHub 一键登录」并保护 /admin 路由

23. 性能优化组合拳  
    图片优化（next/image）、字体子集、Script 策略、动态 import、Webpack 分包、React Compiler  
    任务：把首屏 LCP 从 3.2 s 降到 1.0 s 以内
