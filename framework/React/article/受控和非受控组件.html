概述

什么是受控和非受控组件？

在 React 中，表单组件根据数据管理方式的不同分为两种类型：

  • 受控组件：表单数据由 React 组件状态管理
  • 非受控组件：表单数据由 DOM 自身管理

核心区别

特性受控组件非受控组件
数据存储React 状态DOM 节点
数据流单向数据流双向绑定
更新控制React 控制DOM 控制
实时验证容易实现需要额外处理

受控组件

基本概念

受控组件是指其值由 React 状态控制的表单元素。

每次用户输入时，都会触发状态更新，然后重新渲染组件。

实现原理

import React, { useState } from 'react';

const ControlledComponent = () => {
  const [value, setValue] = useState('');

  const handleChange = (event) => {
    setValue(event.target.value);
  };

  const handleSubmit = (event) => {
    event.preventDefault();
    console.log('提交的值:', value);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input 
        type="text" 
        value={value} 
        onChange={handleChange} 
      />
      <button type="submit">提交</button>
      <p>当前值: {value}</p>
    </form>
  );
};

非受控组件

基本概念

非受控组件是指其值由 DOM 自身管理的表单元素。React 通过 ref 来获取 DOM 节点的值。

实现原理

import React, { useRef } from 'react';

const UncontrolledComponent = () => {
  const inputRef = useRef(null);

  const handleSubmit = (event) => {
    event.preventDefault();
    console.log('输入的值:', inputRef.current.value);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input 
        type="text" 
        ref={inputRef} 
        defaultValue="默认值"
      />
      <button type="submit">提交</button>
    </form>
  );
};

选择指南

使用受控组件的情况：

  • 需要实时验证和格式化输入
  • 需要根据输入动态更新其他 UI
  • 需要实现复杂的表单逻辑
  • 需要表单重置功能
  • 需要即时反馈

使用非受控组件的情况：

  • 简单表单，不需要实时验证
  • 文件上传输入
  • 性能敏感的场景
  • 集成非 React 代码
  • 快速原型开发

const useControlledForm = (initialState = {}) => {
  const [formData, setFormData] = useState(initialState);

  const handleChange = (event) => {
    const { name, value, type, checked } = event.target;
    
    setFormData(prev => ({
      ...prev,
      [name]: type === 'checkbox' ? checked : value
    }));
  };

  const handleMultiSelect = (name, value, isChecked) => {
    setFormData(prev => ({
      ...prev,
      [name]: isChecked
        ? [...(prev[name] || []), value]
        : (prev[name] || []).filter(item => item !== value)
    }));
  };

  const resetForm = () => {
    setFormData(initialState);
  };

  return {
    formData,
    handleChange,
    handleMultiSelect,
    resetForm,
    setFormData
  };
};

表单验证实现

受控组件的实时验证

const ValidatedForm = () => {
  const [formData, setFormData] = useState({
    email: '',
    password: '',
    confirmPassword: ''
  });
  const [errors, setErrors] = useState({});

  const validateField = (name, value) => {
    const newErrors = { ...errors };
    
    switch (name) {
      case 'email':
        if (!value) {
          newErrors.email = '邮箱不能为空';
        } else if (!/\S+@\S+\.\S+/.test(value)) {
          newErrors.email = '邮箱格式不正确';
        } else {
          delete newErrors.email;
        }
        break;
        
      case 'password':
        if (!value) {
          newErrors.password = '密码不能为空';
        } else if (value.length < 6) {
          newErrors.password = '密码至少6位';
        } else {
          delete newErrors.password;
        }
        break;
        
      case 'confirmPassword':
        if (value !== formData.password) {
          newErrors.confirmPassword = '密码不一致';
        } else {
          delete newErrors.confirmPassword;
        }
        break;
    }
    
    setErrors(newErrors);
  };

  const handleChange = (event) => {
    const { name, value } = event.target;
    
    setFormData(prev => ({
      ...prev,
      [name]: value
    }));
    
    // 实时验证
    validateField(name, value);
  };

  const isValid = Object.keys(errors).length === 0;

  return (
    <form>
      <div>
        <input
          name="email"
          type="email"
          value={formData.email}
          onChange={handleChange}
          placeholder="邮箱"
        />
        {errors.email && <span style={{color: 'red'}}>{errors.email}</span>}
      </div>

      <div>
        <input
          name="password"
          type="password"
          value={formData.password}
          onChange={handleChange}
          placeholder="密码"
        />
        {errors.password && <span style={{color: 'red'}}>{errors.password}</span>}
      </div>

      <div>
        <input
          name="confirmPassword"
          type="password"
          value={formData.confirmPassword}
          onChange={handleChange}
          placeholder="确认密码"
        />
        {errors.confirmPassword && (
          <span style={{color: 'red'}}>{errors.confirmPassword}</span>
        )}
      </div>

      <button type="submit" disabled={!isValid}>
        提交
      </button>
    </form>
  );
};

import { useCallback, useRef } from 'react';

const DebouncedInput = () => {
  const [value, setValue] = useState('');
  const timeoutRef = useRef();

  const handleChange = useCallback((event) => {
    const newValue = event.target.value;
    setValue(newValue);

    // 清除之前的定时器
    if (timeoutRef.current) {
      clearTimeout(timeoutRef.current);
    }

    // 设置新的定时器进行防抖
    timeoutRef.current = setTimeout(() => {
      console.log('防抖后的值:', newValue);
      // 这里可以执行搜索或其他昂贵操作
    }, 300);
  }, []);

  return (
    <input
      value={value}
      onChange={handleChange}
      placeholder="输入搜索内容..."
    />
  );
};

// 良好的表单结构
const WellStructuredForm = () => {
  const [formState, setFormState] = useState({
    values: {
      username: '',
      email: '',
      age: ''
    },
    errors: {},
    touched: {},
    isSubmitting: false
  });

  const handleChange = (event) => {
    const { name, value } = event.target;
    
    setFormState(prev => ({
      ...prev,
      values: {
        ...prev.values,
        [name]: value
      },
      // 清除该字段的错误
      errors: {
        ...prev.errors,
        [name]: ''
      }
    }));
  };

  const handleBlur = (event) => {
    const { name } = event.target;
    
    setFormState(prev => ({
      ...prev,
      touched: {
        ...prev.touched,
        [name]: true
      }
    }));
  };

  return (
    <form>
      <input
        name="username"
        value={formState.values.username}
        onChange={handleChange}
        onBlur={handleBlur}
      />
      {formState.touched.username && formState.errors.username && (
        <div>{formState.errors.username}</div>
      )}
    </form>
  );
};

const LoginForm = () => {
  const [credentials, setCredentials] = useState({
    username: '',
    password: ''
  });
  const [errors, setErrors] = useState({});
  const [isSubmitting, setIsSubmitting] = useState(false);

  const handleChange = (event) => {
    const { name, value } = event.target;
    setCredentials(prev => ({
      ...prev,
      [name]: value
    }));
    
    // 清除该字段的错误
    if (errors[name]) {
      setErrors(prev => ({
        ...prev,
        [name]: ''
      }));
    }
  };

  const validateForm = () => {
    const newErrors = {};
    
    if (!credentials.username.trim()) {
      newErrors.username = '用户名不能为空';
    }
    
    if (!credentials.password) {
      newErrors.password = '密码不能为空';
    } else if (credentials.password.length < 6) {
      newErrors.password = '密码至少6位';
    }
    
    setErrors(newErrors);
    return Object.keys(newErrors).length === 0;
  };

  const handleSubmit = async (event) => {
    event.preventDefault();
    
    if (!validateForm()) {
      return;
    }
    
    setIsSubmitting(true);
    
    try {
      // 模拟 API 调用
      await loginAPI(credentials);
      console.log('登录成功');
    } catch (error) {
      setErrors({ submit: error.message });
    } finally {
      setIsSubmitting(false);
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <div>
        <input
          name="username"
          value={credentials.username}
          onChange={handleChange}
          placeholder="用户名"
          disabled={isSubmitting}
        />
        {errors.username && <div>{errors.username}</div>}
      </div>
      
      <div>
        <input
          name="password"
          type="password"
          value={credentials.password}
          onChange={handleChange}
          placeholder="密码"
          disabled={isSubmitting}
        />
        {errors.password && <div>{errors.password}</div>}
      </div>
      
      {errors.submit && <div>{errors.submit}</div>}
      
      <button 
        type="submit" 
        disabled={isSubmitting}
      >
        {isSubmitting ? '登录中...' : '登录'}
      </button>
    </form>
  );
};

总结

核心要点

  1. 受控组件：数据由 React 状态管理，适合需要实时控制和验证的场景
  2. 非受控组件：数据由 DOM 管理，适合简单表单和性能敏感场景
  3. 选择依据：根据具体需求选择合适的方式，也可以混合使用
  4. 性能优化：使用 useCallback、useMemo 和防抖等技术优化性能
  5. 最佳实践：良好的状态结构、错误处理和用户体验

现代趋势

随着 React Hook 的发展，推荐在大多数场景下使用受控组件，结合自定义 Hook 来管理复杂的表单逻辑。对于简单的表单或文件输入，非受控组件仍然是很好的选择。

掌握受控和非受控组件的使用，能够帮助你在不同场景下做出最合适的技术选择，构建出既高效又用户友好的表单体验。
