备孕期的最佳做法

1. 同步进行，而非分阶段

  • 减脂和增肌同步进行（即身体重组，Body Recomposition）
  • 通过合理饮食+抗阻训练，在减少脂肪的同时保持/增加肌肉量

2. 体重目标建议

  • 备孕理想BMI是 18.5-23.9（对应体重约47.5-61kg）
  • 体脂率健康和代谢指标正常

3. 具体执行方案

饮食方面：

  • 创造轻微热量缺口（每天少300-500大卡）
  • 蛋白质充足（1.2-1.6g/kg体重，约85-110g/天）
  • 补充叶酸（备孕必需）、铁、维生素D

运动方面：

  • 每周3-4次中等强度力量训练（深蹲、硬拉、推举等复合动作）
  • 每周2-3次低强度有氧（快走、游泳、椭圆机）
  • 避免过度高强度训练（可能影响排卵）

4. 重要提醒

  • 不要极端节食：可能影响激素水平和排卵周期
  • 不要追求快速减重：每周减0.5-1斤为宜
  • 建议先体检：检查激素水平、甲状腺功能、血糖血脂

5. 何时可以开始备孕？

  • 体重降到130斤左右，且月经周期规律
  • 体脂率降到28-30%以下
  • 医生评估身体状态良好后

备孕期推荐运动体系

一、必做：盆底肌训练（凯格尔）

为什么最重要：

  • 孕晚期子宫增大会严重压迫盆底肌
  • 提前强化可避免产后漏尿、脏器脱垂
  • 产后恢复速度会快很多

做法：

  • 找到盆底肌：排尿时突然憋住的那块肌肉
  • 收缩3-5秒 → 放松3-5秒，10次一组，每天3组
  • 进阶：尝试快速收缩（1秒收缩1秒放松）

二、推荐加入的动作

核心类（替代卷腹，保护腹直肌）：

  • 死虫式 ✓（你已提到）
  • 鸟狗式（四点支撑，对侧手脚伸展）
  • 侧平板支撑（30-45秒）
  • 臀桥（激活臀肌+核心）

下肢力量（为孕期承重做准备）：

  • 深蹲 ✓（你已提到）→ 建议高脚杯深蹲或相扑深蹲（对腰椎压力小）
  • 罗马尼亚硬拉（练臀腿后侧，改善体态）
  • 箭步蹲（单腿稳定性）
  • 侧卧抬腿（强化臀中肌，预防孕期骨盆不稳）

上肢与背部（改善圆肩，为抱娃做准备）：

  • 俯卧撑（跪姿开始）
  • 哑铃划船
  • 面拉（改善圆肩）

低强度有氧：

  • 快走（最容易坚持）
  • 游泳（对关节零压力，孕中晚期也能做）
  • 椭圆机

三、每周安排建议

天数内容
周一下肢力量（深蹲+硬拉+臀桥）
周二低强度有氧30分钟
周三上肢+核心（俯卧撑+死虫+鸟狗）
周四休息或瑜伽/拉伸
周五全身循环训练
周六低强度有氧或游泳
周日完全休息

四、需要避免的运动

  • ❌ 大重量硬拉/深蹲（压迫腰椎）
  • ❌ 仰卧卷腹（孕中晚期绝对禁止，备孕期也建议少做）
  • ❌ 高强度跳跃（如波比跳、跳绳过多）
  • ❌ 高温瑜伽/热普拉提
  • ❌ 竞技性强的运动（防受伤）

五、给你的具体建议

每次训练结构（40-50分钟）：

  1. 热身5分钟
  2. 凯格尔运动（正式训练前激活盆底肌）
  3. 主训练：深蹲 3组×12次 → 臀桥 3组×15次 → 死虫式 3组×10次/侧
  4. 辅助：哑铃划船 3组×12次 → 俯卧撑 3组×力竭
  5. 拉伸5分钟

关键提醒： 备孕/孕期核心原则是保持稳定、避免腹部高压。有任何下腹坠胀感或不适，立即停止并咨询医生。

本周执行检查清单

完整的动作图解说明书，按训练日分类：

📋 周一：下肢力量日

1. 凯格尔运动（激活）

步骤：

  • 找到盆底肌：想象憋尿时用力的肌肉
  • 收缩3-5秒 → 放松3-5秒
  • 不要憋气，正常呼吸
  • 不要收紧腹部/臀部/大腿

2. 臀桥

步骤：

  • 仰卧，双脚踩地，膝盖弯曲90度
  • 呼气时臀部发力向上顶
  • 顶峰收缩1秒（夹紧臀部）
  • 吸气缓慢下落，轻触地面立即再次抬起

❌ 常见错误：用腰顶而不是臀顶（会腰痛）

3. 相扑深蹲

步骤：

  • 双脚打开约两倍肩宽，脚尖外八45度
  • 双手持矿泉水瓶于胸前
  • 膝盖对准脚尖方向下蹲
  • 大腿尽量平行地面，感受大腿内侧拉伸
  • 脚跟不要离地

4. 单腿臀桥

步骤：

  • 基础臀桥姿势，抬起一条腿伸直
  • 保持骨盆水平（不要歪向一侧）
  • 臀部发力顶起，支撑腿脚跟踩实
  • 每侧10次，换边

要点：如果骨盆晃动，说明臀中肌弱，更要练！

5. 侧卧抬腿（练臀中肌）

步骤：

  • 侧卧，下方腿弯曲稳定身体
  • 上方腿伸直，脚尖微内扣（朝地面）
  • 臀部侧面发力抬起腿
  • 感受臀部侧面收紧，不是大腿外侧

6. 靠墙静蹲

步骤：

  • 背部贴墙，双脚前移约50cm
  • 缓慢下滑至大腿平行地面
  • 膝盖不超过脚尖，小腿垂直地面
  • 保持45秒，感受大腿前侧燃烧

📋 周二：核心稳定日

1. 猫牛式（热身）

步骤：

  • 四点跪姿，手腕在肩下，膝盖在髋下
  • 牛式：吸气，塌腰抬头，尾骨上翘
  • 猫式：呼气，拱背低头，肚脐找脊柱
  • 像猫伸懒腰一样流畅交替

2. 死虫式（核心王牌动作）

步骤：

  • 仰卧，手臂上举垂直地面，大腿抬起屈膝90度
  • 呼气时，对侧手脚同时伸展（手向头顶，脚向地面）
  • 关键：腰部始终贴紧地面！如果拱起，说明幅度太大
  • 吸气收回，换边

💡 自检：手伸进腰和地面之间，应该抽不出来

3. 鸟狗式

步骤：

  • 四点跪姿，核心收紧
  • 呼气时，对侧手脚同时伸直（手前伸，腿后蹬）
  • 保持身体稳定，不要左右晃动
  • 想象头顶到脚跟有一条直线
  • 吸气收回，换边

4. 侧平板支撑（膝盖着地版）

步骤：

  • 侧卧，下方膝盖弯曲90度着地，上方腿伸直
  • 手肘在肩膀正下方支撑
  • 臀部抬起，身体成一条直线
  • 保持30秒，感受侧腹发力

进阶：双腿都伸直，脚前后交叉着地

5. 骨盆卷动

步骤：

  • 仰卧，双脚踩地，双手放身体两侧
  • 呼气时，骨盆后倾（腰部贴地，耻骨向上）
  • 吸气时，骨盆前倾（腰部拱起，耻骨向下）
  • 像用骨盆画圆圈，激活深层核心

6. 婴儿式拉伸

步骤：

  • 跪姿，膝盖分开略宽于髋，大脚趾相触
  • 臀部坐向脚跟，身体前倾趴下
  • 手臂向前伸展，额头贴地
  • 深呼吸，放松腰背和髋部

📋 周四：上肢+全身循环

1. 跪姿俯卧撑

步骤：

  • 跪姿，膝盖着地，双手在肩下略宽于肩
  • 身体从膝盖到头顶一条直线（不要塌腰）
  • 屈肘下落，胸口接近地面
  • 推起时呼气，感受胸部发力

❌ 常见错误：只动胳膊不动身体（像磕头）

2. 超人式（俯卧两头起）

步骤：

  • 俯卧，手臂向前伸直，双腿伸直
  • 呼气时，同时抬起双手和双脚
  • 感受下背部和臀部收紧
  • 保持2秒，缓慢落下

注意：不要过度后仰，适度抬起即可

3. 平板支撑

步骤：

  • 手肘在肩膀正下方，前臂平行
  • 双脚踩地，身体成一条直线
  • 收紧腹部和臀部，不要塌腰或撅屁股
  • 保持40秒，正常呼吸不要憋气

💡 想象：头顶向前延伸，脚跟向后蹬，身体像一块钢板

📋 周六：综合功能日

腹式呼吸（必练！）

步骤：

  • 仰卧或坐姿，一手放胸口，一手放肚子
  • 吸气（4秒）：肚子鼓起，胸口不动
  • 呼气（6秒）：肚子收缩，像腰带收紧
  • 呼吸比例：吸:呼 = 1:1.5

备孕意义：学会腹式呼吸是孕晚期和分娩时的重要技能！
